1. Banker’s Algorithm
Q: What is Banker’s Algorithm?
A: It is a deadlock avoidance algorithm that checks safe state before allocating resources.
Q: What is Need matrix?
A: Need = Max – Allocation
Q: What is safe state?
A: A state where all processes can complete without deadlock.
Q: What is safe sequence?
A: Order in which processes execute safely.

🔹 2. Deadlock
Q: What is deadlock?
A: When processes wait forever for resources.
Q: Conditions for deadlock?
A:
Mutual exclusion
Hold and wait
No preemption
Circular wait

🔹 3. Disk Scheduling
(VERY IMPORTANT 🔥 — always asked)
Q: What is disk scheduling?
A: Method to reduce head movement and improve performance.
Q: What is FCFS?
A: First request served first.
Q: What is SSTF?
A: Shortest seek time first.
Q: What is SCAN?
A: Moves like elevator (left → right).
Q: Difference SCAN vs C-SCAN?
A:
SCAN → goes back
C-SCAN → jumps to start
Q: What is LOOK?
A: Stops at last request, not disk end.

🔹 4. Head Movement
Q: What is head movement?
A: Total distance traveled by disk head.

Q: Types of file allocation?
A:
Contiguous
Linked
Indexed
Q: Contiguous allocation advantage?
A: Fast access
Q: Disadvantage?
A: Fragmentation
Q: Linked allocation advantage?
A: No fragmentation
Q: Indexed allocation?
A: Uses index block to store addresses

🔹 6. Bit Vector
Q: What is bit vector?
A: Array showing free/allocated blocks (0 = free, 1 = allocated)

Q: What is MPI?
A: Message Passing Interface used for parallel computing.
Q: What is MPI_Reduce?
A: Combines results (sum, min, max).
Q: What is MPI_Comm_rank?
A: Gives process ID.
Q: What is MPI_Comm_size?
A: Total number of processes.
